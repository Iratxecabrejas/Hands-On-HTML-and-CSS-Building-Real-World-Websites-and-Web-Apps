Table Structure Explained

  • <thead> - This tag is used for the header section of the table. It helps group header rows so they are easy to identify. It is particularly useful when tables are long and may need repeated header information.
  • <tbody> - This tag is used for the main body of the table, containing the data rows. It helps in separating the content rows from the header and footer, making the table more readable and maintainable.
  • <tfoot> - This tag is for the footer section, often used for summary information like totals or averages. It is typically placed after the <tbody> but before closing the <table> tag, allowing the browser to render it effectively even before all data rows are loaded.
  • <th> - Defines a header cell in the table. The text in <th> is usually bold and centered, providing an easy way to differentiate header cells from normal data cells. Use <th> to make your table more accessible as screen readers use these tags to describe the data.
  • <td> - Defines a standard cell containing data in the table. It holds the actual information you want to display. You can use attributes like rowspan or colspan to merge cells as needed.
  • colspan - An attribute used to make a cell span across multiple columns. For example, if you want a "Total" label to span across three columns, you use colspan="3". This is helpful in creating more readable summary or footer rows.
