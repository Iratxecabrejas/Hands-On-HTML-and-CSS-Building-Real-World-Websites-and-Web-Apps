Table Structure Explained

  • <thead> - This tag is used for the header section of the table. It helps group header rows so they are easy to identify.
  • <tbody> - This tag is used for the main body of the table, containing the data rows.
  • <tfoot> - This tag is for the footer section, often used for summary information like totals or averages.
  • <th> - Defines a header cell in the table. The text in <th> is usually bold and centered.
  • <td> - Defines a standard cell containing data in the table.
  • colspan - An attribute used to make a cell span across multiple columns.
